# LLMNews

LLMNews is the start of a minimalist AI and LLM model news web app built around Bun, TanStack Router, Base UI primitives, shadcn-inspired styling, and Evlog-based observability planning.

## Docs
- `/docs/product-overview.md`
- `/docs/tech-stack.md`
- `/docs/implementation-plan.md`

## Getting started
```bash
bun install
bun run dev
```

If Bun is unavailable in your environment, the current scaffold is also compatible with standard Node tooling for installation and builds.
