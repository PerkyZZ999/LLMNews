# LLMNews product overview

## Vision
LLMNews is a minimalist web app focused on surfacing new AI and LLM model releases as early as possible.

## Core user promise
- Track fresh model launches quickly
- Separate signal from hype with short summaries
- Keep an AI-native, lightweight reading experience

## MVP scope
- A landing page that communicates the product direction
- A curated "early signal" feed section for notable releases
- A stack foundation built around Bun, TanStack Router, Base UI primitives, and shadcn-inspired UI patterns
- Structured logging reserved for ingestion and editorial workflows with Evlog

## Next implementation slices
1. Add source ingestion pipelines for labs, model hubs, and research feeds
2. Introduce authentication and saved briefings
3. Add editorial tooling, tagging, and push alerts
